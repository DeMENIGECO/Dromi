=========================
      CARTELLA OPT
=========================

Questa cartella contiene file opzionali (non necessari)
per il corretto funzionamento di Dromi.

Le applicazioni qui presenti sono già installate,
ma sono incluse per chi desidera utilizzarle.

Il file dromi.ico è usato per le icone dei collegamenti
(es. Dromi Docs, DFIE).

-------------------------
FILE CONTENUTI:
-------------------------

- dromi.ico
  → Icona per i collegamenti

- docs_opener.exe
  → Programma per aprire Dromi Docs

- dfie.exe
  → DFIE (Dromi Files Integrated Editor)

-------------------------
INFORMAZIONE EXTRA:
-------------------------

DFIE significa:
Dromi Files Integrated Editor
(Editor Integrato dei File Dromi)

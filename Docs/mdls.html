Passo 1: Installare il primo pacchetto con pip e usarlo

Dromi supporta moduli esterni, installabili tramite DromiPip, oppure per la shell, pip.

Per installare un pacchetto, è sufficiente aprire il terminale e digitare il comando dromi.exe --mdl pip install nome_pacchetto. Ad esempio, per installare il pacchetto "windawk", è sufficiente digitare dromi.exe --mdl pip install windawk.

Installiamo il primo pacchetto:

dromi.exe --mdl pip install windawk

Se tutto è andato bene, dovrebbe essere installato il pacchetto e pronto all'uso.

Create un file windawk.dr e metteteci:

from mdl import windawk 

windawk.windawk()

Salvatelo, e nel terminale eseguite il comando dromi.exe windawk.dr nella cartella dello script per eseguire lo script.

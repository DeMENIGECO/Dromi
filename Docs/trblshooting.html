Risoluzione dei problemi

Il comando dromi.exe non funziona

Se il comando dromi.exe non funziona, prova a usare dromi, oppure se vuoi usare una versione installata recentemente (funziona nei gi installer di Dromi 1.0.2+) usa dromi123, ad esempio, se hai installato Dromi 1.1.0, usa dromi110.

Se nessuno di questi comandi funziona, è possibile che Dromi non sia stato installato correttamente. Prova a reinstallare Dromi e assicurati di seguire tutte le istruzioni.

Il comando dromi.exe funziona, ma non riesco a eseguire gli script

Se il comando dromi.exe funziona, ma non riesci a eseguire gli script, è possibile che tu stia eseguendo il comando nella cartella sbagliata. Assicurati di eseguire il comando nella cartella in cui si trova lo script che vuoi eseguire.

Il comando dromi.exe funziona, ma quando eseguo lo script non succede nulla

Se il comando dromi.exe funziona, ma lo script non produce alcun risultato, prova a salvare il file con CTRL+S (O il menu File > salva)

Il comando dromi.exe funziona, ma quando eseguo lo script ottengo un errore

Se il comando dromi.exe funziona, ma quando esegui lo script ottieni un errore, è possibile che ci sia un errore di sintassi nel tuo script. Controlla attentamente il tuo script per assicurarti che non ci siano errori di battitura o di sintassi.

Se non riesci a risolvere l'errore, prova a cercare l'errore online o a chiedere aiuto su forum o community di programmazione.

Il comando dromi.exe funziona, ma non riesco a usare i moduli installati

Se il comando dromi.exe funziona, ma non riesci a usare i moduli installati, è possibile che tu non stia importando correttamente i moduli nel tuo script. Assicurati di importare i moduli correttamente usando la sintassi corretta.

Se lo hai installato con il comando dromi.exe --mdl pip install nome_del_modulo, al posto di import nome_del_modulo, usa from mdl import nome_del_modulo

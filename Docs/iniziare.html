Passo 1: Installare Dromi

Andate agli installer e scaricate l'ultima versione per le vostre specifiche.

Una volta scaricato, eseguite l'installer e seguite le istruzioni. Dromi è disponibile per Windows.

Se avete eseguito bene le istruzioni, Dromi sarà installato e pronto all'uso, ma per verificare, digitate nel terminale il comando dromi.exe e verificate se funziona.

Passo 2: il primo file

Per creare un file Dromi, è sufficiente creare un file di testo con estensione .dr. Ad esempio, potete creare un file chiamato test.dr.

Una volta creato, potete aprirlo con un editor di testo e scrivere il vostro primo script Dromi. Scriveteci dentro:

set_window_size(600,400) 
set_window_title("Prova Interfaccia") 
h1("Benvenuto in Dromi GUI") 
h2("Test sottotitolo") 
p("Questa è una label normale") 
button("Cliccami") 
dr.init()

Salvatelo, e nel terminale eseguite il comando dromi.exe test.dr nella cartella dello script per eseguire lo script.

Se tutto è andato bene, dovrebbe aprirsi una finestra con il titolo "Prova Interfaccia" e all'interno i vari elementi che avete creato.
